Fizz Buzz: A Coding Challenge

 The point of this program is to switch values in an array. You first select the inputs that you want the array to divide by in the first two, you then can select the bounds in which you want the array to check for.

 Example: 3, 5, 1, 100: will go through the numbers 1-100 and then look for multiple of 3 and 5. (9 will be Fizz, 20 will be Buzz and 15 will be FizzBuzz)
